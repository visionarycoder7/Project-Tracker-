# 🚀 Project Contribution Tracker

A simple yet powerful web application built using **Python (Flask)** and **SQLite** to track individual contributions in group projects.

This platform allows users to submit their work, earn points, and compete on a leaderboard — making teamwork more transparent and accountable.

---

## 📌 Features

* 👤 User Registration
* 📝 Task Submission System
* 🎯 Contribution-Based Scoring
* 🏆 Leaderboard Ranking System
* 📊 Real-time Progress Tracking

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Database:** SQLite
* **Frontend:** HTML (Basic)

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/project-tracker.git
cd project-tracker
```

---

### 2️⃣ Install Dependencies

```
pip install flask
```

---

### 3️⃣ Initialize Database

```
python database.py
```

---

### 4️⃣ Run the Application

```
python app.py
```

---

### 5️⃣ Open in Browser

```
http://127.0.0.1:5000
```

---

## 🎮 How It Works

1. Register a new user
2. Submit tasks with assigned points
3. Points are added to the user’s score
4. Leaderboard updates automatically
